## 游戏玩法

### 移动逻辑

W/A/S/D 控制蛇的移动，每一次刷新只识别第一次对方向的改变

### 得分逻辑

每移动1格获得一分，吃不同食物有对应的分数。

单人模式下：无分数上限，有单机排行榜记录最高分。

多人模式下：先得到2000分者胜，或者有人死亡则另一方胜利。

### 食物种类

- normal食物：蓝色方块，食用之后长度增加1，获得50分。
- chaos食物：紫色方块，食用之后上下左右操控变反，持续10秒，这段时间内得分翻倍。
